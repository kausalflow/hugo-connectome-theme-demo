Graph of Connected Articles

#Tutorial

All articles that are connected by using links in metadata Link Articles Articles can be linked by adding a the reference of the target post to the meta data of the current post, e.g., links: - snm/single-neuron-model.md - tags/gauss-markov-theorem The ref of the target post can be the path to the markdown file inside the content folder, such as snm/single-neuron-model.md, the path to the tag page, such as tags/gauss-markov-theorem. If one needs to find the ref of a post, it is found on the right of the content. will show up in the graph. Articles that are not connected will be hidden.
